# Design System Outline

## Colors

### Light Theme
- **Background**: `#FFFFFF` (white)
- **Text/Stroke**: `#000000` (black)
- **Gradients**: White-based gradients with opacity variations

### Dark Theme
- **Background**: `#1C1C1C` (dark gray)
- **Text/Stroke**: `#FFFFFF` (white)
- **Gradients**: Dark gray-based gradients with opacity variations

## Typography

- **Font Family**: Lato
- **Font Sizes**:
  - **Hero**: 90pt
  - **Scrolling Words**: 16pt
  - **Connect Button**: 20pt
- **Font Weights**:
  - **Hero**: 
    - Light theme: Thinner
    - Dark theme: Thicker

## Spacing & Layout

### Viewport Sizes
- **Desktop**: 1280 × 832px
- **Mobile**: 390 × 844px

### Component Dimensions

#### Buttons
- **Desktop**: 237 × 58px
- **Mobile**: 350 × 70px
- **Border Radius**: 12px (mobile)
- **Stroke Width**: 2px

## Components

### Buttons
- **Style**: Outlined (stroke-based, no fill)
- **Border Radius**: 12px (mobile)
- **Stroke**: 2px width
- **Font Size**: 20pt (Connect button)
- **Colors**: 
  - Light theme: Black stroke
  - Dark theme: White stroke

### Gradients
- Linear gradients used for visual effects
- Opacity variations (0 to 1)
- Used in decorative elements

## Design Tokens

### Border Radius
- **Small**: 12px (used for mobile buttons)

### Stroke Width
- **Standard**: 2px

### Opacity
- **Transparent**: 0
- **Opaque**: 1
- **Gradient stops**: Various (0.4375, 0.5625, etc.)

## Notes

- Design supports both light and dark themes
- Responsive design with mobile and desktop variants
- Minimal color palette focusing on contrast
- Clean, outlined component style
